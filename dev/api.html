dabl API¶

This is a list of all functions and classes provided by dabl.

High-level API¶

clean(X[, type_hints, verbose]) Public clean interface
detect_types(X[, type_hints, …]) Detect types of dataframe columns.
plot_supervised(X, target_col[, type_hints, …]) Exploration plots for classification and regression.
SimpleClassifier([refit, random_state, verbose]) Automagic anytime classifier.
EasyPreprocessor([scale, verbose, types]) A simple preprocessor

Supervised Models¶

SimpleClassifier([refit, random_state, verbose]) Automagic anytime classifier.

EasyPreprocessor([scale, verbose, types]) A simple preprocessor
detect_types(X[, type_hints, …]) Detect types of dataframe columns.
clean(X[, type_hints, verbose]) Public clean interface

Plotting¶

class_hists
discrete_scatter(x, y, c[, legend, …]) Scatter plot for categories.
find_pretty_grid(n_plots[, max_cols]) Determine a good grid shape for n_plots subplots.
mosaic_plot(data, rows, cols[, …]) Create a mosaic plot from a dataframe.
plot_supervised(X, target_col[, type_hints, …]) Exploration plots for classification and regression.
plot_classification_categorical(X, target_col) Exploration plots for categorical features in classification.
plot_classification_continuous(X, target_col) Exploration plots for continuous features in classification.
plot_regression_categorical(X, target_col[, …]) Exploration plots for categorical features in regression.
plot_regression_continuous(X, target_col[, …]) Exploration plots for continuous features in regression.

GridSuccessiveHalving(estimator, param_grid) Search over specified parameter values with successive halving.
RandomSuccessiveHalving(estimator, …[, …]) Randomized search on hyper parameters.
